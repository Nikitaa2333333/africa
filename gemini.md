# Конституция проекта (gemini.md)

## Схемы данных (Payload Shape)
Сайт полностью статический. Результатом (Payload) будет файловая структура:
- `index.html` (Скелет и разметка, интерактивный 3D-глобус, калькулятор, секция проектов с табами, карусель партнеров, форма с выбором направления)
- `css/style.css` (Премиальная светлая тема, стеклянные эффекты Glassmorphism, микро-анимации, адаптивная сетка Grid/Flexbox)
- `js/script.js` (Логика вращающегося 3D-глобуса на Canvas/CSS, калькулятор, плавная прокрутка, анимация появления контента)
- `assets/` (Оптимизированные векторные логотипы, иконки, изображения)
- Итоговый артефакт: ZIP-архив с этой структурой.

## Правила поведения (Behavioral Rules)
- Использовать только чистый HTML5, CSS3 и Vanilla JS (без Next.js/React/TailwindCSS, если нет явного требования).
- Фокус на премиальный дизайн, крутой визуал и идеальную производительность (60fps анимации, быстродействие, доступность).


## Архитектурные инварианты
- Декларативный премиум-дизайн сначала создается и согласовывается через макеты **Pencil**.
- Развертывание осуществляется исключительно переносом ZIP-архива в `public_html/` через File Manager cPanel. Отказ от старого WordPress.
- **Основной фоновый цвет:** Чистый белый (`#FFFFFF`). Для визуального зонирования и структуры используется премиальное чередование с ультрасветлым серым (`#F5F6F7`), на котором белые карточки (`#FFFFFF`) выделяются за счет тонких контрастных рамок и легких теней.
- **Акцентный цвет:** Фирменный красный (`#D81C24`), используемый точечно для кнопок, активных элементов и акцентов на глобусе.
- **Цвет основного текста:** Глубокий графитовый (`#272A2C`) для максимального контраста и читаемости.
- **Цвет вспомогательного текста:** Торговый серый (`#666D71`).
- **Типографика:** Montserrat для акцентных заголовков (в обычном регистре с естественным начертанием), Inter для информационных блоков и данных.
- **Главный визуальный фокус:** Интерактивный вращающийся 3D-глобус на главном экране (реализованный с помощью HTML5 Canvas или CSS 3D), адаптированный под светлую тему (темно-серые точки сетки, красные акцентные торговые пути с движущимися частицами).
